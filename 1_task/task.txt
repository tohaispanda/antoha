Задание 1.

- Представить 3 блока текста. В каждом блоке заголовок 2-го уровня (они должны быть курсивом) и по 3 параграфа.
- Вверху, над всеми блоками будет заголовок 1-го уровня. Он должен быть выделен красным цветом.
- Каждый блок должен быть отделен от друг друга горизонтальной полоской.
- Решение должно содержать полную разметку для HTML документа.

Доп. задание*:

- Сделать для каждого параграфа первую букву больше по шрифту в 2 раза и зеленого цвета используяю CSS.
- Сделать "красную строку" так же с использованием CSS
- Задать стили через атрибут элемента, встроенные стили в документ, через подключенный файл стилей.
